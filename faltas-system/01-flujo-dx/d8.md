# [D8] CIERRE DEL TRÁMITE

## Objeto
ACTA

---

## Convención del bloque

En este bloque debe quedar siempre explícito:

- qué `TipoActo` o resultado administrativo se produce
- qué `TipoDocumento` lo representa, si aplica
- si debe invocarse `P1`
- si requiere notificación formal
- si debe invocarse `P3`
- cuál es la salida del bloque

---

## Finalidad

Cerrar el trámite dentro del sistema de forma:

- explícita
- auditable
- tipificada
- consistente con el estado real del proceso

El cierre:

- finaliza el flujo interno
- registra el motivo
- conserva trazabilidad completa
- puede permitir reingreso controlado

---

## Regla transversal clave

D8:

- NO elimina ni modifica historia
- NO “reabre” el trámite
- SÍ permite reingreso mediante eventos

👉 Modelo append-only

---

## Tipos de cierre (OBLIGATORIO)

Campo:
TipoCierre

## Regla específica del bloque

D8 debe dejar explícito:

- que el cierre siempre tiene `TipoCierre`
- que un cierre puede o no estar respaldado por un `TipoActo`
- que si llega un evento externo no se “reabre” el trámite
- que el sistema **reingresa** mediante un nuevo evento

### Regla de reingreso

D8
→ EventoExterno
→ TipoResultadoExterno
→ D4 o D5 o nuevo D8
